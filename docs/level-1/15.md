## 1. Apa itu Flood Routing?

Flood routing adalah proses memperkirakan perubahan bentuk hidrograf banjir saat aliran bergerak sepanjang sungai atau waduk.

Routing menjawab pertanyaan:
- Berapa debit di hilir?
- Kapan debit puncak terjadi?
- Seberapa besar peredaman banjir?

---

## 2. Konsep Penyimpanan Aliran

Dalam routing sungai:

\[
S = K \left[ X I + (1 - X) O \right]
\]

di mana:
- \(S\) = simpanan air di segmen sungai
- \(I\) = debit masuk
- \(O\) = debit keluar
- \(K\) = parameter waktu perjalanan
- \(X\) = parameter pembobot (0â€“0.5)

---

## 3. Metode Muskingum

Metode Muskingum menggunakan pendekatan diskrit:

\[
O_{t+1} = C_0 I_{t+1} + C_1 I_t + C_2 O_t
\]

Koefisien \(C_0, C_1, C_2\) ditentukan dari parameter \(K, X,\) dan interval waktu.

Metode ini menunjukkan bahwa:
- sungai menyimpan air sementara
- debit puncak di hilir lebih kecil dan terjadi lebih lambat

---

## 4. Interpretasi Fisik Routing

Routing menyebabkan:
- **peredaman debit puncak**
- **pergeseran waktu puncak**
- **pelebaran hidrograf**
