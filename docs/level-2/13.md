## 4. Kapan Model ML Lebih Unggul?

- Data historis banyak
- Sistem sangat kompleks
- Fokus pada prediksi jangka pendek

Kapan model fisik lebih unggul?

- Perlu interpretasi proses
- Perubahan tata guna lahan
- Simulasi skenario masa depan
