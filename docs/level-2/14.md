## 1. Apa itu Sensitivitas Model?

Sensitivitas model menunjukkan seberapa besar perubahan hasil model ketika suatu parameter diubah.

Contoh:
- Perubahan kecil nilai Curve Number dapat mengubah debit puncak secara signifikan.

---

## 2. Sumber Ketidakpastian dalam Model

Ketidakpastian dapat berasal dari:

| Sumber | Contoh |
|-------|--------|
| Data | Data hujan tidak lengkap |
| Parameter | Nilai Manning n tidak pasti |
| Struktur model | Pemilihan metode losses |
| Skala spasial | Resolusi DEM rendah |

Semua model mengandung ketidakpastian.
