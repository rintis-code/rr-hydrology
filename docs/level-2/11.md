## 1. Apa itu Machine Learning?

Machine Learning adalah pendekatan pemodelan yang belajar dari data historis untuk memprediksi nilai di masa depan tanpa eksplisit menggunakan persamaan fisik.

Dalam hidrologi, ML dapat digunakan untuk:
- prediksi debit
- prediksi hujan
- koreksi bias model fisik

---

## 2. Model Fisik vs Model ML

| Aspek | Model Fisik | Model ML |
|------|-------------|----------|
| Dasar | Proses fisik | Pola data |
| Kelebihan | Interpretatif | Fleksibel |
| Kekurangan | Banyak parameter | Kurang transparan |

Model ML tidak menggantikan model fisik, tetapi melengkapinya.

## 4. Risiko Penggunaan ML

ML bisa menghasilkan prediksi yang baik secara statistik tetapi salah secara fisik.

Engineer tetap harus:
- memeriksa kewajaran hasil
- memastikan model tidak melanggar logika hidrologi
