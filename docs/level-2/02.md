## 3. Sumber Data

### a. Data Observasi Lapangan
Keunggulan:
- lebih akurat secara lokal
- representatif kondisi nyata

Keterbatasan:
- sering tidak lengkap
- jangkauan spasial terbatas

---

### b. Data Satelit / Remote Sensing

Keunggulan:
- cakupan luas
- tersedia untuk wilayah tanpa stasiun

Keterbatasan:
- resolusi terbatas
- perlu kalibrasi terhadap data lapangan

---

## 4. Kualitas Data

Data harus diperiksa:

- apakah ada data hilang?
- apakah ada nilai ekstrem tidak wajar?
- apakah periode data cukup panjang?

Kualitas data menentukan kepercayaan terhadap model.
