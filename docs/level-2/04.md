## 1. Apa itu Kalibrasi?

Kalibrasi adalah proses menyesuaikan parameter model agar hasil simulasi mendekati data observasi.

Tujuannya:
- model merepresentasikan kondisi nyata DAS
- parameter tetap masuk akal secara fisik

Kalibrasi bukan sekadar â€œmencocokkan grafikâ€, tetapi memastikan parameter tetap realistis.

---

## 2. Apa itu Validasi?

Validasi adalah proses menguji model dengan data yang **tidak digunakan saat kalibrasi**.

Tujuannya:
- memastikan model tidak hanya cocok untuk satu periode
- menguji konsistensi performa model

Jika model hanya bagus saat kalibrasi tetapi buruk saat validasi â†’ model overfitting.

## 4. Overfitting dan Underfitting

### Overfitting
Model terlalu â€œdipaksaâ€ mengikuti data kalibrasi sehingga gagal pada data lain.

### Underfitting

Engineer harus mencari keseimbangan antara keduanya.
