## Tujuan Modul

Setelah mempelajari modul ini, mahasiswa mampu:

1. Memahami hubungan antara model hidrologi dan hidraulika.
2. Menggunakan output hidrograf dari HEC-HMS sebagai input ke HEC-RAS.
3. Menjelaskan alur sistem pemodelan banjir terpadu.
4. Menginterpretasikan dampak perubahan debit terhadap muka air sungai.

---

## 1. Mengapa Perlu Integrasi Model?

HEC-HMS menghasilkan **debit banjir**, tetapi tidak menunjukkan:

- tinggi muka air
- area genangan
- dampak pada penampang sungai

HEC-RAS menggunakan debit tersebut untuk menghitung respon hidraulika sungai.

---

## 2. Alur Integrasi Model

\[
\text{Hujan} \rightarrow \text{HEC-HMS (Debit)} \rightarrow \text{HEC-RAS (Muka Air)}
\]

Langkah umum:
1. Jalankan model HEC-HMS
2. Ekspor hidrograf debit di outlet
3. Masukkan hidrograf sebagai boundary condition di HEC-RAS
4. Jalankan simulasi aliran tak mantap

---

## 3. Tantangan Integrasi

Beberapa masalah yang sering muncul:

- perbedaan interval waktu
- perbedaan satuan
- kesalahan posisi titik input debit
- debit terlalu besar atau kecil untuk geometri sungai

Engineer harus memastikan konsistensi antara dua model.

---

## 4. Makna Fisik Integrasi

Integrasi memungkinkan engineer melihat:

- bagaimana hujan ekstrem memengaruhi tinggi muka air
- potensi limpasan dari sungai
- risiko banjir di titik tertentu

Tanpa integrasi, analisis banjir tidak lengkap.
