## 1. Apa itu Artificial Neural Network?

ANN adalah model Machine Learning yang meniru cara kerja jaringan saraf manusia untuk menemukan pola dalam data.

Struktur dasar ANN terdiri dari:
- Input layer (misalnya data hujan)
- Hidden layer (proses pembelajaran pola)
- Output layer (misalnya debit sungai)

---

## 2. ANN dalam Prediksi Debit

ANN dapat digunakan untuk mempelajari hubungan:

\[
\text{Hujan (t, t-1, t-2)} \rightarrow \text{Debit (t)}
\]

Model belajar dari data historis untuk memprediksi debit pada waktu berikutnya.

---

## 3. Kelebihan dan Keterbatasan ANN

**Kelebihan:**
- Mampu menangkap hubungan non-linear
- Tidak perlu persamaan fisik eksplisit

**Keterbatasan:**
- Membutuhkan data yang cukup banyak
- Sulit diinterpretasikan secara fisik
- Risiko overfitting

---

## 4. Peran Engineer dalam Model ANN
