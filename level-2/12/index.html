    12. Neural Network untuk Prediksi Debit

    Posisi Modul

    Level 2 â€” Hydrology with Engineering Tools, ML & AI
    Blok B: Advanced Modeling + ML/AI Integration

    Catatan: Laporan tugas menggunakan format IMRAD sesuai panduan Level 2.

    Tujuan Modul

    Setelah mempelajari modul ini, mahasiswa mampu:

    1. Memahami konsep dasar Artificial Neural Network (ANN).
    2. Menjelaskan bagaimana ANN digunakan untuk prediksi debit.
    3. Menyusun model ANN sederhana berbasis data hujan dan debit.
    4. Mengevaluasi hasil prediksi ANN secara statistik dan fisik.

    1. Apa itu Artificial Neural Network?

    ANN adalah model Machine Learning yang meniru cara kerja jaringan saraf manusia untuk menemukan pola dalam data.

    Struktur dasar ANN terdiri dari: - Input layer (misalnya data hujan) - Hidden layer (proses pembelajaran pola) - Output layer (misalnya debit sungai)

    2. ANN dalam Prediksi Debit

    ANN dapat digunakan untuk mempelajari hubungan:

    [ \text{Hujan (t, t-1, t-2)} \rightarrow \text{Debit (t)} ]

    Model belajar dari data historis untuk memprediksi debit pada waktu berikutnya.

    3. Kelebihan dan Keterbatasan ANN

    Kelebihan: - Mampu menangkap hubungan non-linear - Tidak perlu persamaan fisik eksplisit

    Keterbatasan: - Membutuhkan data yang cukup banyak - Sulit diinterpretasikan secara fisik - Risiko overfitting

    4. Peran Engineer dalam Model ANN

    Engineer harus memastikan:

    • data input relevan
    • hasil prediksi masuk akal secara hidrologi
    • model tidak hanya cocok pada data latihan

    ANN adalah alat bantu, bukan pengganti pemahaman proses hidrologi.

    • membantu memahami struktur ANN
    • membantu membuat skrip pelatihan model
    • membantu memplot hasil prediksi

    AI tidak boleh menggantikan: - pemilihan variabel input - interpretasi hasil prediksi debit

    Mahasiswa harus menjawab:

    1. Mengapa ANN bisa lebih fleksibel daripada model fisik?
    2. Apa risiko overfitting pada ANN?
    3. Mengapa hasil ANN harus diuji secara fisik, bukan hanya statistik?

    Latihan

    1. Jelaskan fungsi hidden layer pada ANN.
    2. Mengapa data historis penting dalam pelatihan ANN?
    3. Apa perbedaan ANN dan regresi linier biasa?
