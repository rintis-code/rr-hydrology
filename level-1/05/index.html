    05. Data Hujan: Jenis, Alat Ukur, Uji Data, dan Hujan Rerata

    CPL & Posisi dalam RPS

    • CPL: CPL-04 (Problem-Solving)
    • Tahap RPS: 5
    • Fokus: memahami sumber data hujan, kualitas data, serta cara menentukan hujan rerata wilayah

    Tujuan Modul

    Setelah mempelajari modul ini, mahasiswa mampu:

    1. Menjelaskan jenis data hujan dan sumbernya.
    2. Mengidentifikasi alat ukur hujan yang umum digunakan.
    3. Memahami pentingnya uji kualitas data hujan.
    4. Menghitung hujan rerata wilayah dengan metode sederhana.

    1. Jenis Data Hujan

    Data hujan dapat dibedakan berdasarkan:

    ðŸ”¹ Berdasarkan Waktu

    • Data harian
    • Data jam-jaman
    • Data hujan maksimum tahunan

    ðŸ”¹ Berdasarkan Sumber

    • Stasiun hujan manual
    • Stasiun hujan otomatis (AWS/ARR)
    • Data hujan satelit (misalnya GPM, CHIRPS)

    Dalam hidrologi teknik, pemilihan jenis data tergantung pada: - tujuan analisis (banjir, irigasi, neraca air), - skala waktu yang dibutuhkan.

    2. Alat Ukur Curah Hujan

    Beberapa alat pengukur hujan:

    ðŸ”¹ Ombrometer (Manual Rain Gauge)

    • Pengukuran dilakukan secara manual
    • Umumnya menghasilkan data harian

    ðŸ”¹ Automatic Rain Gauge (ARG)

    • Mencatat hujan secara otomatis
    • Dapat menghasilkan data intensitas hujan

    Keakuratan alat dan perawatan stasiun sangat memengaruhi kualitas data.

    3. Permasalahan Umum pada Data Hujan

    Data hujan sering mengalami:

    • Data hilang (missing data)
    • Nilai ekstrem yang tidak wajar
    • Perubahan lokasi atau lingkungan stasiun
    • Kesalahan pencatatan

    Karena itu, sebelum digunakan, data harus melalui uji kualitas (data quality control).

    4. Uji Kualitas Data Hujan (Konsep Dasar)

    Beberapa konsep uji data hujan yang perlu dipahami:

    ðŸ”¹ Konsistensi Data

    Mengecek apakah data masih berada dalam pola yang wajar dibanding periode lain atau stasiun sekitar.

    ðŸ”¹ Kelengkapan Data

    Persentase data yang tersedia dalam suatu periode.

    ðŸ”¹ Deteksi Nilai Ekstrem

    Nilai yang jauh berbeda dari pola umum perlu diverifikasi.

    Pada tahap ini mahasiswa memahami pentingnya uji data, perhitungan rinci dapat dibahas pada modul lanjutan.

    5. Hujan Rerata Wilayah

    Karena hujan tidak merata secara spasial, diperlukan metode untuk menentukan hujan rerata suatu DAS.

    Metode sederhana yang diperkenalkan:

    ðŸ”¹ Rata-rata Aritmatik

    [ P_{avg} = \frac{P_1 + P_2 + \dots + P_n}{n} ]

    Cocok jika: - sebaran stasiun merata, - topografi relatif seragam.

    Metode lain (akan dibahas lebih lanjut): - Poligon Thiessen - Isohyet

    6. Peran Data Hujan dalam Analisis Hidrologi

    Data hujan digunakan untuk:

    • analisis hujan rancangan
    • perhitungan limpasan
    • pemodelan hujanâ€“debit
    • analisis kekeringan

    Kesalahan pada data hujan akan langsung memengaruhi hasil analisis hidrologi.

    Contoh Konseptual

    Jika terdapat 3 stasiun hujan dalam DAS dengan data:

    Stasiun Hujan (mm)
    A 80
    B 100
    C 90

    Maka hujan rerata sederhana:

    [ P_{avg} = \frac{80 + 100 + 90}{3} = 90 \text{ mm} ]

    Latihan Pemahaman

    1. Mengapa uji kualitas data hujan penting sebelum digunakan dalam analisis hidrologi?
    2. Sebutkan perbedaan utama antara stasiun hujan manual dan otomatis.
    3. Hitung hujan rerata jika empat stasiun mencatat 75 mm, 82 mm, 90 mm, dan 88 mm.

    Output yang Dikumpulkan

    Mahasiswa mengumpulkan: - jawaban latihan, - satu tabel sederhana perhitungan hujan rerata wilayah.
