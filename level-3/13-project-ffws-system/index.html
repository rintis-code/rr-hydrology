    13. Proyek Besar: Perancangan Sistem FFWS Berbasis Agent AI

    Posisi Modul

    Level 3 â€” Expert Hydrology (AI Agents)

    Modul ini adalah proyek utama Level 3, di mana mahasiswa merancang sistem Flood Forecasting Warning System (FFWS) sebagai sistem keputusan berbasis model dan Agent AI.

    Tujuan Proyek

    Mahasiswa mampu:

    1. Merancang arsitektur sistem FFWS terintegrasi.
    2. Menghubungkan model hidrologi, hidraulika, dan sistem tindakan.
    3. Mengintegrasikan Agent AI sebagai decision-support.
    4. Mengevaluasi sistem dari sisi teknis, risiko, dan dampak.

    1. Deskripsi Proyek

    Mahasiswa diminta merancang sistem FFWS untuk satu wilayah studi (nyata atau hipotetik) yang memiliki risiko banjir.

    Sistem harus mencakup:

    • monitoring hidrologi
    • model prediksi
    • analisis dampak
    • sistem peringatan
    • opsi tindakan teknis
    • peran Agent AI

    Fokus utama adalah arsitektur sistem, bukan perhitungan ulang model dasar.

    2. Komponen yang Harus Dirancang

    a. Monitoring System

    • Data hujan
    • Tinggi muka air
    • Debit sungai

    b. Forecasting Layer

    • Model hidrologi
    • Model hidraulika
    • (opsional) model ML

    c. Impact Assessment

    • Area terdampak
    • Kedalaman genangan
    • Risiko terhadap infrastruktur dan penduduk

    d. Decision Layer

    • Ambang batas peringatan
    • Opsi tindakan teknis
    • Integrasi Agent AI untuk rekomendasi

    e. Warning & Response

    • Level peringatan
    • Tindakan operasional
    • Koordinasi antar pihak

    3. Peran Agent AI dalam Proyek

    Agent AI harus dirancang untuk:

    • membandingkan beberapa skenario prediksi
    • mengevaluasi risiko relatif
    • memberi rekomendasi tindakan

    Namun mahasiswa harus menunjukkan bahwa: keputusan akhir tetap dilakukan oleh manusia.

    4. Fokus Penilaian Proyek

    Penilaian didasarkan pada:

    âœ” Logika decision layer
    âœ” Pemahaman risiko & dampak
